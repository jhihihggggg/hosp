║  🔄 UPDATE AFTER CODE CHANGES                                 ║
╚════════════════════════════════════════════════════════════════╝

1. Push changes to GitHub (from your dev environment):
   
   git add -A
   git commit -m "Your changes"
   git push origin main

2. Update on VPS:
   
   cd /var/www/hosp
   chmod +x vps_update.sh
   sudo ./vps_update.sh

║  🔧 TROUBLESHOOTING                                           ║
╚════════════════════════════════════════════════════════════════╝

View Application Logs:
    sudo journalctl -u nazipuruhs -f

Check Service Status:
    sudo systemctl status nazipuruhs

Restart Application:
    sudo systemctl restart nazipuruhs

Check Nginx:
    sudo systemctl status nginx
    sudo tail -f /var/log/nginx/error.log

Recreate Users (if login issues):
    cd /var/www/hosp
    source venv/bin/activate
    python manage.py shell < create_production_accounts.py
